# Rishabh Kala Portfolio

This is my personal portfolio website, built to showcase my work as an AI Platform Engineer and Full-Stack Product Builder.

The website highlights my experience with AI-agent platforms, workflow automation, admin dashboards, RBAC and audit systems, API gateways, model management, scheduler and notification systems, and cloud-based backend infrastructure.

## What's Included

- About me
- Professional experience
- Technical skills
- Focus areas
- Education
- Contact details
- Downloadable resume
- GitHub and LinkedIn links

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Lucide React

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in the browser to view the portfolio.

## Build

Create a production build:

```bash
npm run build
```

Start the production server:

```bash
npm run start
```
